# Ageda Arcade Smoke Test

This repo contains a small functional test for AGEDA v5.3.1.

## Games

1.  **[Snake](snake.html)**: Classic snake game. Control with arrows, reset with space.
2.  **[Arcade Smoke](smoke.html)**: Interactive particle simulation. Move mouse to emit smoke, click to cause an explosion.

## Local Development

Just open the `.html` files in any modern web browser.
