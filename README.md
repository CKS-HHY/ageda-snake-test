# Snake Arcade - AGEDA Single-Chain Layer Propagation Smoke Test

A polished, playable Snake game built with HTML5 Canvas, CSS3, and JavaScript.

## Features
- Classic Snake gameplay.
- Arcade-style neon visuals.
- High score tracking via `localStorage`.
- Responsive keyboard controls (Arrow Keys or WASD).

## How to Run
1. Clone this repository.
2. Open `snake.html` in any modern web browser.
3. Use Arrow keys or WASD to start moving and eat the red food.
